# CI/CD for Me

## What

```html
<p>
  CI/CD คือ
  รูปแบบของการทำงาน(workflow)ที่บูรณาการให้มีการทำงานร่วมกันอย่างต่อเนื่อง(pipeline)
</p>
<p>
  และเป็นการทำงานแบบอัตโนมัติ(Automation)โดยแบ่งหน้าที่ในการทำงาน(stage)ออกเป็นแต่ละหน้าที่
</p>
เช่น test, build, deploy เป็นต้น
```

## Why

```
เพราะว่าการใช้ CI/CD มีการตรวจสอบตามขั้นตอนที่ได้เตรียมไว้ก่อนส่งมอบ\ss
และการทำงานที่ทำได้อย่างต่อเนื่อง อัตโนมัติ ทำให้ลดความผิดพลาดที่จะเกิดขึ้น
```
